# Crusader Kings 3 Developer-Diary REPLY Threads — Markdown Corpus

*Generated 2026-06-10 by extract_replies.py (companion to the dev-diary corpus).*

This corpus is the **community-voice counterpart** to the dev-diary corpus. For each dev diary, it captures the full reply thread — every reply post across every forum page — as one provenance-tagged Markdown file (`rt_*.md`) structured to mirror `dd_*.md`. Reply bodies are preserved verbatim; quoted text and reaction tallies are wrapped in `<!-- artifact:* -->` comments and excluded from word counts, so the two corpora can be distant-read and compared on equal terms.

Each `rt_*.md` front-matter carries `parent_dd_file` linking it to its dev diary, plus `reply_count`, `participant_count`, `pages_captured`, `reply_date_first/last`, and `quoted_span_count`.
